# AIVOA CRM AI Assistant

An AI-powered CRM interaction logging system built for AIVOA assignment.

## 🚀 Features

- AI Assistant for logging HCP interactions
- LangGraph-based multi-tool agent
- Auto-fill CRM form from natural language input
- Save interactions to SQLite database
- View saved records API
- React + FastAPI full stack architecture

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- CSS

### Backend
- FastAPI
- Python
- SQLite
- SQLAlchemy

### AI / Agent
- Groq LLM
- LangGraph

## 🤖 LangGraph Tools

1. Log Interaction  
2. Edit Interaction  
3. Summarize Notes  
4. Sentiment Analyzer  
5. Follow-up Generator
